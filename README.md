🚨 Women Safety Alert System



A simple web-based emergency alert system that allows users to send their live location instantly in case of danger.



* Features



\- One-click emergency button

\- Real-time location tracking

\- Alert sound for immediate attention

\- Email notification with location details



* Tech Stack



\- Python (Flask)

\- HTML, CSS (Bootstrap)

\- JavaScript (Geolocation API)



* How It Works



1\. User clicks the \*SEND HELP\* button  

2\. The system gets the user’s current location  

3\. An alert sound is triggered  

4\. An email with location details is sent to a predefined contact
